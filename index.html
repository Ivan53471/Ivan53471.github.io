Go调用geth JSON RPC接口

发表于 2024-03-27 分类于 科研
本文字数： 2k 阅读时长 ≈ 2 分钟

在Go中调用以太坊的格式如下

有以下常用接口

net

1、net_version
当前连接网络的ID

2、net_listening
客户端是否处于监听状态

eth

1、eth_accounts
返回账户地址数组

2、eth_getBalance
需要传入账户地址，返回余额，单位是wei
“latest” 表示你要查询的是当前链上最新的账户余额

3、eth_coinbase
获取挖矿账户地址

4、eth_mining
返回客户端是否在挖矿

5、eth_getTransactionCount
返回指定地址发生的交易数量

6、eth_blockNumber
获取当前块编号

personal

1、personal_listAccounts
获取该节点下的所有账户

2.personal_newAccount
创建用户，需要将密码作为参数传入

3.personal_lockAccount
锁定指定账户，需要将地址作为参数传入

4.personal_unlockAccount
解锁指定账户,有些操作必须解锁账户才可以操作

db

1.db_putString
在本地数据库中存入字符串

2.db_getString
从本地数据库读取字符串

Linux系统构建以太坊私链

发表于 2024-03-26 更新于 2024-03-27 分类于 科研
本文字数： 1.1k 阅读时长 ≈ 1 分钟

配置Go环境

1、首先切换到管理员身份
sudo -s

2、下载官⽅的 go 语⾔编译器压缩包
wget https://go.dev/dl/go1.21.3.linux-amd64.tar.gz

3、解压缩刚下载的新版本并放置到合适的位置
tar -C /usr/local -xzf go1.21.3.linux-amd64.tar.gz
（如果有旧版本，先输入 rm -rf /usr/local/go 移除）

4、设定环境变量
export PATH=$PATH:/usr/local/go/bin
试试看是否正常（应该会看到版本号）
go version

构建geth

1、下载git
sudo apt install git

2、克隆以太坊源码
这一步很可能连不上github，没什么解决方法，多试
git clone https://github.com/ethereum/go-ethereum.git

3、进入ethereum所在文件夹
cd go-ethereum

4、编译geth
make geth
如果发现下载很慢，可以终止掉，设置代理
export GOPROXY=https://goproxy.io,direct
下载完成，应该可以看到以下画面

5、进⼊⽬录
cd build/bin

6、启动主程序，连接主⽹并同步
./geth

搭建私链

1、创造配置文件
在不同于以太坊源码的文件路径下创一个文件夹和.json文件，命名为genesis.json

2、初始化链

3、启用链
此处启用私链，并且开启RPC接口
geth --datadir ../project/MyChain/ --identity "fanone" --rpc --rpcport 8545 --rpcapi "db,eth,net,web3,personal" --rpccorsdomain "*" --networkid 15
-–identity “fanone”：节点身份标识
-–rpc：开启 RPC 接口
–rpcport：指定端口
–rpcapi：开启的RPC API
–rpccorsdomain：跨域请求，”*”为允许所有来源的跨域请求
–networkid：网络标识符，用于连上私链，networkid是在genesis.json中确定的
