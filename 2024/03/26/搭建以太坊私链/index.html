3、启用链
此处启用私链，并且开启RPC接口

  • identity “MyETH”：节点身份标识
  • rpc：开启 RPC 接口
  • rpcport：指定端口
  • rpcapi：开启的RPC API
  • rpccorsdomain：跨域请求，”*”为允许所有来源的跨域请求
  • networkid：网络标识符，用于连上私链，networkid是在genesis.json中确定的
    console：打开geth控制台，用于创建节点和部署智能合约

  • http.port：指定端口
  • http.api：开启的RPC API
  • http.corsdomain：跨域请求，”*”为允许所有来源的跨域请求
