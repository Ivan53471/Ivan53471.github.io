Go调用geth JSON RPC接口

在Go中调用以太坊的格式如下

有以下常用接口

net

1、net_version
当前连接网络的ID

2、net_listening
客户端是否处于监听状态

eth

1、eth_accounts
返回账户地址数组

2、eth_getBalance
需要传入账户地址，返回余额，单位是wei
“latest” 表示你要查询的是当前链上最新的账户余额

3、eth_coinbase
获取挖矿账户地址

4、eth_mining
返回客户端是否在挖矿

5、eth_getTransactionCount
返回指定地址发生的交易数量

6、eth_blockNumber
获取当前块编号

personal

1、personal_listAccounts
获取该节点下的所有账户

2.personal_newAccount
创建用户，需要将密码作为参数传入

3.personal_lockAccount
锁定指定账户，需要将地址作为参数传入

4.personal_unlockAccount
解锁指定账户,有些操作必须解锁账户才可以操作

db

1.db_putString
在本地数据库中存入字符串

2.db_getString
从本地数据库读取字符串
